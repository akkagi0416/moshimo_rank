# Moshimo ranking app

[もしも](http://www.moshimo.com/)の売り上げランキングを表示

## 方法

[もしもAPI](http://www.moshimo.com/pdf/moshimo_api.pdf)を使ってデータを取得します。


## 取得データ

次のデータを毎日取得

* カテゴリーごと
 * フード・ドリンク・スイーツ
 * ファッション
 * AV・デジモノ
 * 家電
 * 美容・コスメ
 * ダイエット・健康
 * 生活用品・インテリア・雑貨
 * スポーツ・レジャー
 * ホビー・エトセトラ
* JUNゼミ方式
* 全商品
